# 🏋️ PrimeFit Gym Website

A multi-page gym website built using **only HTML5** to practice and strengthen my understanding of semantic HTML, accessibility, forms, tables, and other core HTML concepts.

This project is part of my Full Stack Development learning journey. Rather than focusing on design, I'm focusing on writing clean, well-structured, and meaningful HTML. As I learn new technologies, I'll continue improving this project by adding styling, interactivity, and backend functionality.

## 🎯 Project Goals

* Practice semantic HTML5.
* Build a realistic multi-page website.
* Learn proper webpage structure.
* Improve accessibility.
* Master HTML forms and tables.
* Create a strong foundation before learning CSS and JavaScript.
